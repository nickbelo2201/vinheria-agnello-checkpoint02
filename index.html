Bem vindo!

Este projeto apresenta um sistema de cadastro e interação com vinhos da Vinheria Agnello, desenvolvido utilizando JavaScript. O sistema permite ao usuário escolher tipos de vinhos e interagir com um cardápio, simulando um processo simples de seleção e consulta de estoque. Para organizar o código e evitar repetições, foram utilizadas funções, responsáveis por separar cada parte do sistema, como escolha de vinhos e verificação de estoque. Além disso, estruturas de repetição (como loops) são aplicadas para permitir múltiplas interações com o usuário, tornando o sistema mais dinâmico. O projeto tem como objetivo praticar conceitos fundamentais como reutilização de código, organização lógica e controle de fluxo.
