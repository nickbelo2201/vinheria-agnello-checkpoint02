🍷 Vinharia Agnello

Sistema interativo de uma vinheria desenvolvido em JavaScript, com funcionalidades de cadastro, escolha de vinhos e simulação de controle de estoque.

📌 Sobre o projeto

O projeto simula o funcionamento básico de uma vinheria, permitindo que o usuário:

Faça um cadastro inicial
Escolha entre diferentes tipos de vinhos (Tinto, Branco e Rosé)
Consulte opções disponíveis
Receba informações sobre estoque e classificação dos vinhos
Interaja com o sistema múltiplas vezes

Toda a interação acontece via prompt, alert e console.log.

⚙️ Funcionalidades
🧾 Cadastro de usuário
🍷 Seleção de vinhos por categoria:
Tinto
Branco
Rosé
📊 Classificação dos vinhos por idade:
Jovem
Amadurecido
Antigo
⚠️ Alerta de estoque baixo
🔁 Loop de interação (usuário pode continuar comprando)
📈 Relatório final com:
Total de interações
Vinhos com estoque baixo
Vinho mais antigo
🧠 Lógica aplicada

O projeto utiliza conceitos fundamentais de programação:

Funções para organização do código
Estruturas condicionais (if/else)
Estruturas de repetição (while)
Manipulação de strings
Separação de responsabilidades (cada função com um papel)
🛠️ Tecnologias utilizadas
HTML
CSS
JavaScript
🚀 Como executar
Clone o repositório
Abra o projeto no VS Code
Execute o arquivo index.html no navegador
Interaja com o sistema através dos alertas
🎮 Como usar
Digite seu nome no cadastro
Escolha o tipo de vinho
Selecione uma opção disponível
Veja informações de estoque e classificação
Continue ou finalize a interação
📊 Exemplo de funcionamento

O sistema:

Classifica automaticamente o vinho com base no ano
Exibe alertas caso o estoque esteja baixo
Armazena dados para gerar um relatório final
